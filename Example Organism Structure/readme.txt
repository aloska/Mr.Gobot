Пример папок и файлов некоего организма ИИ

расширения файлов
=================
.genes - набор генов. Может быть для: нейронов, рецепторов, преффекторов, входных и выходных данных
.neurons - тела нейронов
.data - входной или выходной файл
.receptors - тела рецепторов
.preffectors - тела преффекторов
.chemical - файлы синапсов, в которых происходит основная работа, это самые большие файлы, ОНИ ДОЛЖНЫ ИМЕТЬ УНИКАЛЬНЫЙ НОМЕР!!!

онтогенез
=========
При старте в организме должны быть описаны все гены. Онтогенез сформирует из генов остальные необходимые файлы.


Папка Senses - чувства
======================
Прямо в папке может лежать некое синаптическое поле (файл .chemical), и тогда обязаны быть гены нейронов и сами нейроны.

Папка Input-XXX - вход
----------------------
Некий вход в организм.
GenData.genes - обязательно ген данных (только один!) и этот файл содержит только один ген.
GenReceptor-XXX.genes - хотя бы один ген рецепторов
Файлы данных, тела рецепторов генерятся из генов.
Нейроны и гены нейронов необязательно есть, но если есть - то обязательно есть синапсы.

Папка Brain - мозг
==================
Содержит ядра мозга

Папка Core-XXX - ядро
---------------------
Должно быть хотя бы одно описание генов нейронов
